# Wolf & Sheep AI Simulation

An AI simulation where a wolf hunts sheep, 
built to demonstrate FSM and Behaviour Tree implementations in Unity.
Press play and watch the AI in action — no player input required.

## AI Systems Implemented
- **Finite State Machine (FSM)** — wolf state management
- **Behaviour Tree** — sheep state management

## Wolf Behaviour
- Idle state
- Search for sheep
- Chase target
- Attack

## Sheep Behaviour
- Wander
- Flee from wolf
- Seek food when nearby

## Built With
- Unity
- C#
